# ArchFlow

**Generate professional software architecture layers and visual workflows from any idea.**

ArchFlow is a lightweight web application that helps developers and product thinkers quickly visualize the layered architecture and main user workflow of their software ideas.

**Live Demo:** https://gemcancers-debug.github.io/arch-flow

**Repository:** https://github.com/gemcancers-debug/arch-flow

---

## How to Use

1. Enter your software idea title and short description.
2. Select the target platform.
3. Add key features (separated by comma).
4. Click **Generate Architecture**.

The app will automatically create:
- Layered architecture breakdown (Frontend, Business Logic, Data Layer, etc.)
- Visual workflow diagram using Mermaid
- Recommended tech stack
- Step-by-step development roadmap

---

## Rules & Technical Notes
- Fully client-side and works offline.
- All generation happens in your browser.
- Works great on Android and desktop.
- Can be installed as PWA on Android.
